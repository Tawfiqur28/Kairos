# Kairos

This application utilizes several AI models from ModelScope to provide personalized career guidance.

## Models Used

- **ModelScope Qwen-Max**: Used for complex reasoning tasks such as generating detailed career match explanations and creating personalized action plans.
- **ModelScope Qwen-ASR**: Powers the voice-to-text functionality for journal entries, allowing users to speak their thoughts.
- **ModelScope qwen-2.5-7b-instruct**: A smaller, efficient model used for simpler tasks like extracting career themes from the user's profile.
